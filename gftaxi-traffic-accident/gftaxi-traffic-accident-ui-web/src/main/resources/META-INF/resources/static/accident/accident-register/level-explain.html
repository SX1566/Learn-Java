事故等级划分为财产损失事故、伤人事故和死亡事故 3 类。
事故等级越高，代表事故越严重，按最高级别来定性。
死亡事故级别高于伤人事故，伤人事故级别高于财产损失事故。
1.财产损失事故：(指无造成人员伤亡的事故)
重大事故 事故等级 自车损失(万元) 车损总额(万元)
(自车加三者车)
否 财产 1 级 少于 1 万元 少于 3 万元
否 财产 2 级 大于等于 1 万元并且少于 3 万元 大于等于 3 万元并且少于 5 万元
是 财产 3 级 大于等于 3 万元 大于等于 5 万元
2.伤人事故：(指有人伤但未造成人亡的事故)
重大事故 事故等级 任一人医疗费(万元)
否 人伤 1 级 少于 1 万元
是 人伤 2 级 大于等于 1 万元并且少于 3 万元
是 人伤 3 级 大于等于 3 万元
注：按任一人医疗费而不是所有人医疗费的合计来计算。
2.死亡事故：(指造成人亡的事故)
重大事故 事故等级 死亡人数
是 死亡 1 级 1 人
是 死亡 2 级 2 人
是 死亡 3 级 3 人或以上
